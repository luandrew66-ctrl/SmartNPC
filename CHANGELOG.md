# Changelog

## v1.0.0 — Initial Release

### Added
- Core framework: BehaviorTree, StateMachine, ProximityManager
- AI integration: LLMBridge (OpenAI + Anthropic), PromptBuilder, ResponseParser
- Memory system: ConversationLog (session), MemoryStore (persistent via DataStore)
- Utilities: RateLimiter, TokenCounter, Logger
- 5 built-in templates: QuestGiver, Shopkeeper, Guard, Narrator, Companion
- 3 example games: Quest Giver, Shop NPC, Adaptive Enemy
- Full documentation and getting started guide
- Rojo project support
